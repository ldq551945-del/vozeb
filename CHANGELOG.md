- [二开] Vozeb 作为二开版本接入，基于原开源项目 `basketikun/infinite-canvas` 继续开发，并保留 AGPL-3.0 开源协议与原作者致谢。

- [新增] 管理员后台改为侧边栏切换布局，支持概览、系统设置、用户管理和公共提示词库分区管理。

- [新增] README 重写为 Vozeb 项目说明，补充无限画布、AI 创作、素材管理、WebDAV、本地 Agent 与 Codex App 插件等当前功能。

- [优化] 后台侧栏选中态适配深色主题，避免点击后出现突兀反白。
- [修复] 检查更新逻辑兼容本地版本高于远端版本的情况，避免未推送新版本时把远端旧版本显示为最新。
- [修复] 版本记录解析同时支持 `+ [类型]` 和 `- [类型]` 条目，避免格式化后本地 changelog 不显示。
