# CHANGELOG

## Unreleased

- [调整] 项目版本更新为 `v1.0.0`。

- [优化] 生图工作台、视频工作台、画布和素材库上传入口支持从电脑拖拽添加图片、视频或音频素材，并保留原有点击上传、剪贴板上传和生成流程。

- [优化] 后台新增/编辑接口改为默认只填写接口名称、Base URL、API Key 和一键检测接口，高级配置默认收起，避免新手配置时被复杂字段干扰。
- [优化] 接口一键检测会自动补齐模型列表，并回填文本模型、图片模型、视频模型、接口协议、创建路径、查询路径、结果字段、状态字段、参考素材规则和支持时长。
- [优化] 接口配置页按检测结果显示文本、生图、图生图、视频、图生视频、参考视频/音频的简单结论。
- [修复] 后台接口健康检测和系统代理视频类型识别补齐异步视频兼容路径，避免视频接口检测不准或视频请求未按视频类型处理。
- [修复] 兼容常见异步视频任务接口，补齐创建任务和查询结果流程，避免上游已完成但前端一直停留在生成中。
- [修复] 视频请求按接口配置发送时长、比例、清晰度和参考素材字段，避免旧兼容字段和新字段混用造成参数冲突。
- [修复] 图生视频参考素材只提交接口可访问地址；本地 `blob` 或不可访问素材会直接提示，避免上游扣费后按纯文生视频生成。
- [修复] URL 型视频接口的本地参考图会自动转存为临时公开地址，再作为 `referenceImages` 提交给上游；未配置公网域名时继续拦截并提示。
- [调整] 视频时长按接口配置的支持范围校验，不再固定套用旧兼容接口的 5/10/15 秒档位。
- [修复] 接口高级配置的协议、创建路径和查询路径进入实际视频生成链路，避免一键检测保存后生成仍按旧兼容路径请求。
- [修复] 接口高级配置的参考素材规则进入图片任务运行时，图生图可按接口说明选择 multipart 文件上传、JSON 参考图或公网参考图 URL。
- [优化] 兼容视频接口支持使用高级配置的请求字段模板，模板里的模型、提示词、时长、比例、清晰度和参考素材会按当前生成参数自动替换。
- [修复] 兼容视频接口优先按高级配置的结果字段和状态字段读取上游响应，避免自定义接口已完成但前端仍停留在生成中。
- [修复] 图片工作台图生图恢复兼容接口的 multipart 参考图上传，同时保留 URL 型参考图接口规则，避免不同接口的参考图提交方式互相影响。
- [修复] JSON 图生图接口不再被后台通用参考图规则误判为 multipart，图生图会按配置的参考图字段提交；后台一键检测也会回填正确的参考素材规则。
- [优化] 后台高级配置新增通用 JSON 图生图协议选项，检测到对应接口后会保存该协议，避免换域名后图生图参考图提交方式再次被误判。
- [修复] 上游返回内部媒体地址时按当前接口源重写同一路径，避免前端代理 404 或一直停留在生成中。
- [调整] JSON 图生图仅在参数格式错误时切换请求体，遇到 5xx/524 不再自动换请求体重发，减少重复扣费风险。
